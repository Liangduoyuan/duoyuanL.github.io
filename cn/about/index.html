个人说明书

你好，我是梁多原Phil，金牛座，上升天秤，月亮双子，某大师说我要等风来，然后抓住风，我的路子就打开了，不知真假，但现在晴天无风。

E人，万物好奇者，咖啡重度爱好者，聊天爱好者。

我喜欢研究中国古代哲学、文化、政治制度；喜欢自我认知的书籍；喜欢折腾各种新软件，体验各种新功能，优化自己工作流；喜欢骑动感单车（想去考个单车教练证）；关注数码圈；关注AI圈。

我尽力把生活和工作分开，用两种性格和处事方式去对待。

生活中我是一个什么样的人？ #

  1. 我很好奇，喜欢学习新东西，几乎一切我不明白且有趣的东西。我好奇自然科学，想用逻辑理解背后原理，我好奇人文，想用感性理解人性，我好奇宗教，想用缺陷理解信仰。Anyway，好奇心是源动力，也是生活有趣的源泉。我现在学会了苏东坡说的“博观而约取”，意思是兴趣广泛但是不求完全理解，懂得核心要点或者触动自己的点即可。
  2. 我喜欢做分享，甚至是好为人师。我信奉利他主义，相信孔子所说的“己欲立而立人，己欲达而达人”，所以对于需要帮助且在自我努力的人，我会无偿的伸出援手。但有时候助人会觉得很心寒。以下是常见情况：
    1. 说不清楚问题、看到有些人提问就是两三句话说完，其他人一般看不懂他的问题，只能靠读心术。有时候我就是虚荣心作祟（想让人夸），忍不住去搭话帮一把，至少要交流好几轮才知道他想说啥，真的太折磨人。
    2. 不想努力。有些人不想努力，我说了解决方案，他就问这是什么，怎么弄。我都说了方案，当然是他自己解决了。我不想把饭一口口喂给他，我只想帮会用勺子的人。不想努力的经典表现是能搜索到的问题都拿来问。
    3. 中途失踪。聊着聊着人不见了，前面我帮他的努力都白费。更可恶的是去问他弄好没，他就说好了，好了也不告诉人。
    4. 所以现在我看到别人的问题，或者有人找我帮忙，我都得衡量一下这个人是不是伸手党，必要时遏制自己的助人冲动。
  3. 我自我调整能力很快。小时候我妈常对我说吃亏是福，小时候一直以为说的是别人让自己吃亏，后面才发现大部分情况是自己让自己吃亏。吃亏本身事小，从吃亏当中快速调整并且吸取教训就是福。所以现在遇到生活或者工作中的变故，我能很快调整心态沉着应对，I believe there is alway a better way!。
  4. 我的共情力很强。还是跟从小的教育环境相关，我妈总教我做人要包容，所以现在我都会设身处地的站在他人的角度思考问题，即便他人在做有损我利益的事。这一点在人际交往中会让人觉得很真诚，也帮助我结识了一批要好的朋友。
  5. 我情绪稳定，极少情况下会情绪化。一旦我发现自己情绪不好，我会抓紧把手上事情忙完，然后转移注意力调整状态，必要时会对正接触的人说明白，避免影响他人。
  6. 我办事思前顾后。做事前我会把方案想清楚，并且不止一种方案，甚至把善后措施都想好，大到进行一个项目，小到出门买个菜。这种方式能提高办事的稳定性，但是有时会降低行动力。我正在逐渐改变这种观点，往干中学方向靠，想到什么就先去干，事先做的方案往往赶不上变化，过程中优化方案才是效率高的方式。
  7. 我是一个好的倾听者。我很愿意听别人的故事，有时听到相似经历的故事，还会想象是另一个平行时空的自己。我的共情力让我大部分情况能够get到他人的痛点，必要时指出问题核心并给解决方案，但大部分时候我认为别人只是需要有个人说话，并不需要被给出建议，所以好好听就行了。
  8. 勇于承认自己错误。原来我听到有人说我不是，会下意识出口去反驳，后面我认识到我错了，自己怎么看自己和别人怎么看自己同样重要，先去认真听别人的看法，有则改之，无则权当耳边风，不反驳下回才有机会再次听到别人的看法，有利于自己长期的提升。

工作中我是一个什么样的人？ #

  1. 我的适应性很强，在不要求很强的专业知识的前提下可以配合干很多工作，我会快速学习mentor教的东西，很多时候给个方向即可，我可以自行搜索学习，我很珍惜和mentor交流的机会，工作流程以外的剩下的才是别人的精髓，值得好好学习。
  2. 我喜欢和人沟通。一是源于我有不懂的地方需要想别人请教，二是即便别人不如我，也一定有他的特色，同样值得学习。沟通很重要的一种收获是如果在和别人沟通时，自己说出了原来自己没想到的话，这就是一次有意义的沟通。
  3. 我是自律性和环境相关。在充满激情的环境中，稍微自我push一下，我可以做到质量和效率并进，所谓的遇强则强。在闲适的环境中，我能做到一般效率输出，但自我驱动力下降，甚至会随波逐流。如果我进入状态，我的效率比一般人快2-3倍。
  4. 我耐心很足。可以持续地推进一个项目，但我同样希望工作中能够和同伴有简洁明确高效的沟通。适度的优化语言是可以的，过度的长篇大论会消耗耐心。
  5. 我是一个准时的人。我认为准时是对别人的尊重，不管别人有没有做到，但是自己要做到。如果是一起合作或者工作，希望相互尊重。
  6. 我办事需要系统性，条理性很强。我会借助各种软件规划日程、记录进度、归纳资料、反思总结。
  7. 工作需要高度动脑的时候我的黄金时间每天大约在3-4小时，且间断性需要放松休息，如果不休息输出质量会成阶梯式下降。对于程序化工作，我几乎可以做到高强度一直工作而且速度很快，睡一觉刷新疲惫感，输出质量小幅度波动。
  8. 如果我是上级，我会希望自己的下属学会倾听并思考反馈，在理解我所说的基础上，提出想法甚至是反驳，从而达到商量的目的，而不是固执己见，油盐不进。

我的雷区 #

  • 我的雷区1：不要恶意欺骗我！
  • 我的雷区2：约定好时间，就按照计划实施，不要到了时间才临时改计划，这非常耽误事情且消耗人的耐心。
  • 我的雷区3：别装逼，除非你很有本事，不管这本事是自己挣的还是别人给的。

原来的我只懂爱别人，而不懂爱自己，现在的我明白只有自己舒服了，才有能量爱别人。
